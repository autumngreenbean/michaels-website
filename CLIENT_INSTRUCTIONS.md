## ðŸ“… Managing Events

### To Add an Event:

1. Go to the **"EVENTS"** tab
2. Find an empty row
3. Fill in:
   - **Column A**: Event name/title
   - **Column B**: Venue name (this becomes a Google Maps link automatically!)
   - **Column C**: Date (e.g., "JULY 13" or "July 13, 2026")
   - **Column D**: Time (e.g., "4PM" or "4:00 PM")
   - **Column E**: Any additional notes (optional)
4. Save

**Example:**
```
Event: SUNDAY AFTERNOON JAZZ
Location: FOXTROT
Date: JULY 13
Time: 4PM
Additional Notes: (leave blank or add details)
```

### To Edit an Event:

1. Find the event in the EVENTS tab
2. Click the cell and edit
3. Save

### To Remove an Event:

1. Right-click the row number
2. Select "Delete row"
3. Save

**Tip:** Past events can be deleted or moved to keep your event list current!

## ðŸŽ¥ Managing Portfolio Videos

### To Add a Video:

1. Go to the **"Biography Discography Events Video Links"** tab
2. Scroll down to row 25 or below (the video section)
3. In an empty row:
   - **Column B**: YouTube Video ID
   - **Column C**: Video title/description
4. Save

**How to Find Your YouTube Video ID:**

Your YouTube link looks like this:
```
https://www.youtube.com/watch?v=zpt7ffA5-Wc
```

The Video ID is the part after `v=`:
```
zpt7ffA5-Wc
```

Or from a short link:
```
https://youtu.be/zpt7ffA5-Wc
```

The Video ID is after `youtu.be/`:
```
zpt7ffA5-Wc
```

**Copy just the Video ID** (the random letters and numbers) into column B!

### To Reorder Videos:

1. Click on the row number to select the entire row
2. Drag it up or down to reorder
3. The videos on your website will appear in the same order as the sheet

### To Remove a Video:

1. Find the video row
2. Right-click the row number
3. Select "Delete row"
4. Save

## ðŸ†˜ Troubleshooting

### "My changes aren't showing up!"

1. Wait 5 minutes - changes are cached for performance
2. Try a hard refresh: `Ctrl+Shift+R` (PC) or `Cmd+Shift+R` (Mac)
3. Check that you're editing the correct tab
4. Make sure you saved (Google Sheets usually auto-saves)

### "The location links aren't working for events!"

- Make sure you entered the venue name in Column B (Location)
- The link is generated automatically from whatever text you enter

### "Videos aren't showing up!"

- Double-check you copied the Video ID correctly (not the full URL)
- Make sure the Video ID is in Column B and the title is in Column C
- Verify the video is public on YouTube (not private or unlisted)

### "I accidentally deleted something important!"

1. Click **File** > **Version history** > **See version history**
2. Find a version from before you deleted it
3. Click **Restore this version**
